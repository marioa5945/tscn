# tscn

> 1、convention over configuration
> 2、concise yet expressive

## Installation

```sh
# Locally in your project
yarn add tscn --dev

npm install -D tscn

# Or globally
yarn global add tscn

npm install -g tscn
```

## Usage

### Shell

```sh
# Execute a script as `node` + `tsc`.
tscn script.ts [-w]
```

### Debug

```sh
npm run debug
```

### watch

watch files: ./server/\*\*(.ts | .js | .mjs)
restart delay: 1500ms
