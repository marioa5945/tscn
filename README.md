# tscn

## Introduction

## Installation

```sh
# Locally in your project
npm install -D tscn

# Or globally
npm install -g tscn
```

## Usage

### Shell

```sh
# Execute a script as `node` + `tsc`.
ts-node script.ts
```

### Debug

```sh
npm run debug
```
